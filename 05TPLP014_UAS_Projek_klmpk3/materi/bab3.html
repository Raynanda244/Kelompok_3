Membuat website interaktif dengan JavaScript

3.1 Variabel & Tipe Data

JavaScript memiliki 3 cara deklarasi variabel: var, let, dan const.

// Deklarasi variabel
var namaLama = "Ini var"; // Function scope
let namaBaru = "Ini let"; // Block scope
const PI = 3.14; // Konstanta, tidak bisa diubah

// Tipe data
let string = "Hello World"; // String
let number = 42; // Number
let boolean = true; // Boolean
let array = [1, 2, 3]; // Array
let object = {nama: "John", usia: 25}; // Object
let kosong = null; // Null
let tidakTerdefinisi = undefined; // Undefined

// Type checking
console.log(typeof string); // "string"
console.log(typeof number); // "number"
console.log(typeof boolean); // "boolean"
console.log(typeof array); // "object"
console.log(Array.isArray(array)); // true

3.2 Event & DOM Manipulation

Event digunakan untuk menangani interaksi pengguna.

Event Listener

// Mengakses elemen DOM
const button = document.getElementById('myButton');
const output = document.querySelector('.output');

// Menambahkan event listener
button.addEventListener('click', function() {
    output.textContent = 'Tombol diklik!';
    output.style.color = 'green';
});

// Event mouse
element.addEventListener('mouseover', function() {
    this.style.backgroundColor = 'yellow';
});

element.addEventListener('mouseout', function() {
    this.style.backgroundColor = '';
});

// Event keyboard
input.addEventListener('keyup', function(event) {
    console.log('Key pressed:', event.key);
});

// Event form
form.addEventListener('submit', function(event) {
    event.preventDefault(); // Mencegah reload
    // Proses form
});

3.3 AJAX & Fetch API

Fetch API untuk mengambil data dari server tanpa reload halaman.

// Fetch API dasar
fetch('https://api.contoh.com/data')
    .then(response => response.json())
    .then(data => {
        console.log(data);
        // Proses data
    })
    .catch(error => {
        console.error('Error:', error);
    });

// Dengan async/await
async function getData() {
    try {
        const response = await fetch('https://api.contoh.com/data');
        const data = await response.json();
        return data;
    } catch (error) {
        console.error('Error:', error);
    }
}

// POST request
fetch('https://api.contoh.com/data', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        nama: 'John',
        usia: 25
    })
});
