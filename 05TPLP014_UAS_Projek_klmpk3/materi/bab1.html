Bab 1: Pengenalan HTML

Mengenal dasar-dasar HTML untuk membangun struktur website

1.1 Struktur Dokumen HTML

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur halaman web.

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Contoh Halaman HTML</title>
</head>
<body>
    <h1>Selamat Datang</h1>
    <p>Ini adalah paragraf pertama.</p>
</body>
</html>

Elemen Penting:

  • <!DOCTYPE html> - Deklarasi versi HTML
  • <html> - Elemen root
  • <head> - Informasi metadata
  • <body> - Konten utama halaman

1.2 Element HTML Dasar

HTML terdiri dari berbagai element yang memiliki fungsi berbeda-beda.

Heading (h1-h6)

Digunakan untuk judul dengan level berbeda.

<h1>Judul Utama (Level 1)</h1>
<h2>Sub Judul (Level 2)</h2>
<h3>Sub Judul (Level 3)</h3>

Paragraf dan Format Teks

<p>Ini adalah paragraf biasa.</p>
<p><strong>Teks tebal</strong></p>
<p><em>Teks miring</em></p>
<p><u>Teks bergaris bawah</u></p>

List (Daftar)

<!-- List tak berurutan -->
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

<!-- List berurutan -->
<ol>
    <li>Langkah pertama</li>
    <li>Langkah kedua</li>
    <li>Langkah ketiga</li>
</ol>

Gambar dan Link

<!-- Link -->
<a href="https://www.contoh.com">Kunjungi Website</a>

<!-- Gambar -->
<img src="gambar.jpg" alt="Deskripsi gambar" width="300">

1.3 Formulir HTML

Formulir digunakan untuk mengumpulkan input dari pengguna.

<form action="/submit" method="POST">
    <label for="nama">Nama:</label>
    <input type="text" id="nama" name="nama" required>
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>
    
    <label for="password">Password:</label>
    <input type="password" id="password" name="password" minlength="8">
    
    <label for="pesan">Pesan:</label>
    <textarea id="pesan" name="pesan" rows="4"></textarea>
    
    <button type="submit">Kirim</button>
</form>

Demo Formulir Interaktif

Input Types HTML5:

  • type="email" - Validasi email otomatis
  • type="date" - Pilih tanggal
  • type="number" - Input angka
  • type="color" - Pilih warna
  • type="range" - Slider
