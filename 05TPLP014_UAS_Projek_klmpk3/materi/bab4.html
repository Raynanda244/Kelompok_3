4.1 Pengenalan Framework CSS

Framework CSS adalah kumpulan kode CSS yang sudah dibuat sebelumnya untuk mempercepat pengembangan website.

Mengapa Menggunakan Framework?

  • Konsistensi: Desain yang seragam di seluruh halaman
  • Efisiensi: Menghemat waktu pengembangan
  • Responsif: Sudah mendukung mobile-first design
  • Komponen Siap Pakai: Button, form, navbar, dll.
  • Komunitas: Dukungan dan dokumentasi yang luas

4.2 Perbandingan Framework

Framework Kelebihan Kekurangan Penggunaan
Bootstrap Dokumentasi lengkap, komponen siap pakai, komunitas besar File size besar, desain "terlihat seperti Bootstrap" Aplikasi admin, dashboard, startup MVP
Tailwind CSS Utility-first, customisasi tinggi, file size kecil Learning curve, banyak class di HTML Custom design, design system, proyek besar
Bulma Modern, hanya CSS (no JS), sintaks sederhana Komunitas lebih kecil, kurang populer Proyek kecil, preferensi minimal JS
Foundation Enterprise-ready, aksesibilitas baik, fleksibel Learning curve tinggi, dokumentasi kurang Enterprise, aplikasi kompleks

Utility-first CSS framework untuk custom design yang cepat.

4.5 Memilih Framework yang Tepat

Pertimbangan Pemilihan

Pilih Bootstrap jika:

  • Ingin cepat launch MVP
  • Tim familiar dengan Bootstrap
  • Butuh banyak komponen siap pakai
  • Proyek admin/dashboard
  • Prioritas konsistensi & dokumentasi

Pilih Tailwind jika:

  • Ingin custom design unik
  • Tim suka utility classes
  • Proyek besar dengan design system
  • Prioritas performance & file size
  • Mau menghindari "look Bootstrap"

Decision Tree

Q: Apakah proyek membutuhkan design custom yang unik?

→ Ya: Tailwind CSS

→ Tidak, butuh cepat: Bootstrap

→ Tidak, minimal JS: Bulma

4.6 Implementasi Framework

Setup Bootstrap

<!-- Method 1: CDN (Cepat) -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>

<!-- Method 2: NPM (Proyek Besar) -->
npm install bootstrap

// Di file JavaScript
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap';

<!-- Method 3: Download -->
Download dari getbootstrap.com

Setup Tailwind CSS

<!-- Method 1: CDN Play (Development) -->
<script src="https://cdn.tailwindcss.com"></script>

<!-- Method 2: NPM (Production) -->
npm install -D tailwindcss
npx tailwindcss init

// tailwind.config.js
module.exports = {
  content: ["./src/**/*.{html,js}"],
  theme: {
    extend: {},
  },
  plugins: [],
}

// style.css
@tailwind base;
@tailwind components;
@tailwind utilities;
