Bab 2: CSS Dasar

Mengenal Cascading Style Sheets untuk styling website

2.1 Selektor CSS

Selektor digunakan untuk memilih elemen HTML yang akan diberi style.

/* Selektor elemen */
p {
    color: blue;
}

/* Selektor class */
.important {
    font-weight: bold;
}

/* Selektor ID */
#header {
    background-color: #333;
}

/* Selektor kombinasi */
div.container p {
    margin: 10px;
}

/* Selektor pseudo-class */
a:hover {
    color: red;
}

/* Selektor atribut */
input[type="text"] {
    border: 1px solid #ccc;
}

2.2 Layout CSS

CSS menyediakan berbagai teknik untuk membuat layout website.

Flexbox

Flexbox untuk layout satu dimensi (baris atau kolom).

.container {
    display: flex;
    justify-content: space-between;
    align-items: center;
    gap: 1rem;
}

.item {
    flex: 1;
    padding: 1rem;
}

Grid untuk layout dua dimensi (baris dan kolom).

.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);

2.3 Responsive Design

Membuat website yang tampil optimal di berbagai ukuran layar.

Media Queries

/* Mobile first */
.container {
    padding: 1rem;
}

/* Tablet */
@media (min-width: 768px) {
    .container {
        padding: 2rem;
    }
}

/* Desktop */
@media (min-width: 1024px) {
    .container {
        max-width: 1200px;
        margin: 0 auto;
    }
}

Unit CSS yang Responsif

  • % - Persentase relatif terhadap parent
  • vw/vh - Viewport width/height
  • rem - Relatif terhadap root font-size
  • em - Relatif terhadap font-size parent

2.4 Animasi & Transition

Membuat website lebih hidup dengan animasi CSS.

Transition

.box {
    width: 100px;
    height: 100px;
    background-color: blue;
    transition: all 0.3s ease;
}

.box:hover {
    width: 150px;
    background-color: red;
    transform: rotate(15deg);
}

Animation dengan @keyframes

@keyframes bounce {
    0%, 100% { transform: translateY(0); }
    50% { transform: translateY(-20px); }
}

.animated-box {
    animation: bounce 1s infinite;
}
